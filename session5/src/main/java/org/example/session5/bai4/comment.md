Tốc độ ban đầu :
(A) Copy-Paste (Lặp lại mã nguồn) : Nhanh, chỉ cần Ctrl+C/Ctrl+V.
(B) Layout Dialect (Kế thừa giao diện) : Tốn thời gian cấu hình Bean và tách file ban đầu.


Bảo trì :
(A) Copy-Paste (Lặp lại mã nguồn) : Cực hình. Nếu đổi logo, bạn phải sửa thủ công ở tất cả các trang.
(B) Layout Dialect (Kế thừa giao diện) : Dễ dàng. Chỉ cần sửa 1 lần duy nhất tại file layout.html.


Rủi ro :
(A) Copy-Paste (Lặp lại mã nguồn) : Dễ dẫn đến sai lệch (trang này có Header, trang kia quên Footer).
(B) Layout Dialect (Kế thừa giao diện) : Đảm bảo tính nhất quán (Consistency) 100% cho toàn bộ hệ thống.


Độ sạch của Code :
(A) Copy-Paste (Lặp lại mã nguồn) : Code bị "phình" to bởi những đoạn mã lặp đi lặp lại.
(B) Layout Dialect (Kế thừa giao diện) : Code trang con rất gọn, chỉ tập trung vào nội dung chính.
